# Patchseq Human Neocortex

![Patchseq Human Neocortex screenshot](/assets/papers/HMBAViewers/PatchSeq_Human.webm)

**Project:** [HMBA:Basal Ganglia Preprint/Patchseq Human Neocortex](https://allen-hmba-releases.s3.amazonaws.com/cytosplore/Release/BasalGanglia_Preprint/Preprint_2025_Patchseq_Human_Neocortex.mv) in the [Cytosplore Viewer](/#get)

**Contact:** Julian
**Email:** j.g.l.thijssen@lumc.nl

**Description:** This project provides interactive, linked visualizations of Human Neocortex Patch-seq data gathered by the HMBA project. It integrates three single-cell modalities—transcriptomics, electrophysiology, and morphology—enabling users to explore how these features relate to one another and uncover what makes different cell types distinct.
