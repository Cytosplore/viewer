# Archive

<br/>
**Cytosplore Viewer 3.4.0** &emsp; <i class="fab fa-windows"></i> Windows
&emsp; [Installer (39MB)](https://sec.lumc.nl/mtg-viewer/viewer/win/3.4.0/install_cytosplore_viewer.exe)
&emsp; <i class="fab fa-apple"></i> MacOS 10.12 or higher
&emsp; [Installer (39MB)](https://sec.lumc.nl/mtg-viewer/viewer/mac/CytosploreViewer.3.4.0.dmg)
* Added a spatial transcriptomics mode for data from the SpaceTx Consortium.

**Cytosplore Viewer 3.0.0** &emsp; <i class="fab fa-windows"></i> Windows
&emsp; [Installer (39MB)](https://sec.lumc.nl/mtg-viewer/viewer/win/3.0.0/install_cytosplore_viewer.exe)
&emsp; <i class="fab fa-apple"></i> MacOS 10.12 or higher
&emsp; [Installer (39MB)](https://sec.lumc.nl/mtg-viewer/viewer/mac/CytosploreViewer.3.0.0.dmg)

- Added multi-species comparison mode for data from the BICCN
  - Primary motor cortex (M1) of Human, Marmoset and Mouse
  - Previously avaialble in the Cytosplore Comparison Viewer.
- MacOS version (10.12 or higher)

**Cytosplore Viewer 2.0.0** &emsp; <i class="fab fa-windows"></i> Windows
&emsp; [Installer (38MB)](https://sec.lumc.nl/mtg-viewer/viewer/win/2.0.0/install_cytosplore_viewer.exe)

- Added October 2019 data releases from the Allen Cell Types Database:
  - Cell Diversity in the Human Cortex
  - Cell Diversity in the Mouse Cortex and Hippocampus
- Performance Improvements:
  - Faster t-SNE map computations using GPU technology
  - Faster differential expression computations
  - Reduced loading times due to a new fileformat

**Cytosplore Viewer 1.0.1** &emsp; <i class="fab fa-windows"></i> Windows
&emsp; [Full Installer (300MB)](https://sec.lumc.nl/mtg-viewer/viewer/win/1.0.1/install_cytosplore_viewer.exe)
&emsp; [Update Installer (30MB)](https://sec.lumc.nl/mtg-viewer/viewer/win/1.0.1/update_cytosplore_viewer.exe)

- Cytosplore MTG Viewer renamed to Cytosplore Viewer
- Now supports two datasets:
  - Human Middle Temporal Gyrus (MTG)
  - Mouse Primary Visual Cortex (VISp) and Anterior Lateral Motor Area (ALM)

**Cytosplore Viewer 1.0** &emsp; <i class="fab fa-windows"></i> Windows
&emsp; [Full Installer (300MB)](https://sec.lumc.nl/mtg-viewer/viewer/win/1.0/install_cytosplore_mtg.exe)
&emsp; [Update Installer (30MB)](https://sec.lumc.nl/mtg-viewer/viewer/win/1.0/update_cytosplore_mtg.exe)

- Initial Release of Cytosplore MTG Viewer.
