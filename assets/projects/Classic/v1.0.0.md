# Mouse Neocortex Viewer

**ID:** classic-mouse-neocortex-viewer

![Classic Neocortex Screenshot](/assets/papers/ClassicViewers/mouse.gif)

**Project:** Mouse Neocortex in the [Classic Viewer](/documentation/releases#classic-viewer-install)

**Contact:** Jeroen
**Email:** j.eggermont@lumc.nl

**Description:** Interactive viewer of mouse neocortex single‑cell transcriptomics. Shows 23,822 cells from V1 and ALM, the 133 transcriptomic types, the shared GABAergic versus area‑restricted glutamatergic patterns, and links types to projection properties.

- [Publication](https://www.nature.com/articles/s41586-018-0654-5)
