**Description:** The re-embedding viewer provides interactive exploration of RNA‑seq data from the HMBA Basal Ganglia Consensus Cell Type Atlas. It links UMAP and multiscale HSNE embeddings with the consensus taxonomy across human, macaque, and marmoset. Users can compute differential expression between arbitrary subsets and re‑embed selected data using UMAP, t‑SNE, HSNE, or PCA.
