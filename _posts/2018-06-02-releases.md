Here you can find the releases and change notes of Cytosplore Viewer. If you need an older build for reproducing previous results please download them below.

&emsp; [Download Installer (38MB)](https://sec.lumc.nl/mtg-viewer/win/2.0.0/install_cytosplore_viewer.exe)

**Version 2.0.0** &emsp; <i class="fab fa-windows"></i> Windows
&emsp; [Installer (38MB)](https://sec.lumc.nl/mtg-viewer/win/2.0.0/install_cytosplore_viewer.exe)
* Added October 2019 data releases from the Allen Cell Types Database:
  * Cell Diversity in the Human Cortex
  * Cell Diversity in the Mouse Cortex and Hippocampus
* Performance Improvements:
  * Faster t-SNE map computations using GPU technology
  * Faster differential expression computations
  * Reduced loading times due to a new fileformat
  
**Version 1.0.1** &emsp; <i class="fab fa-windows"></i> Windows
&emsp; [Full Installer (300MB)](https://sec.lumc.nl/mtg-viewer/win/1.0.1/install_cytosplore_viewer.exe)
&emsp; [Update Installer (30MB)](https://sec.lumc.nl/mtg-viewer/win/1.0.1/update_cytosplore_viewer.exe)
* Cytosplore MTG Viewer renamed to Cytosplore Viewer
* Now supports two datasets:
  * Human Middle Temporal Gyrus (MTG)
  * Mouse Primary Visual Cortex (VISp) and Anterior Lateral Motor Area (ALM)
