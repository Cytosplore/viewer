# Simian Viewer

**ID:** simianevo-simian-viewer

![Simian Viewer demo](/assets/papers/Simian_Viewer.gif)

**Project:** [Other/Simian Viewer](https://sec.lumc.nl/mtg-viewer/MV_data/CytosploreHMBAViewer/SimianViewer/SimianViewerProject.mv) in the [Cytosplore Viewer](/#get)

**Contact:** Soumyadeep (s[dot]basu[at]lumc[dot]nl)

**Description:** Cytosplore Simian Viewer provides interactive visualization of multi-species single-cell transcriptomic datasets. It links cell type annotations and gene expression across human, macaque, and marmoset, enabling comparative analysis of cellular and molecular features. Users can explore species-specific and conserved gene expression patterns, analyze cell type similarities and differences, and derive evolutionary insights.
