# Human Mouse Marmoset Motorcortex Viewer

**ID:** classic-hmm-motorcortex-viewer

![Classic HMM Screenshot](/assets/papers/ClassicViewers/hmm.gif)

**Project:** Comparison between Human, Marmoset and Mouse in the [Classic Viewer](/documentation/releases#classic-viewer-install)

**Contact:** Jeroen
**Email:** j.eggermont@lumc.nl

**Description:** Interactive viewer of M1 transcriptomic and epigenomic profiles across human, marmoset, and mouse, featuring a >450,000‑nuclei cross‑species atlas with consensus cell‑type classification, species‑specific specializations (proportions, gene expression, methylation, chromatin), and links to physiology/morphology (e.g., Betz cells).

- [Publication: Nature](https://www.nature.com/articles/s41586-021-03465-8)
