# ATACseq Marmoset Subcortex

**ID:** hmba-atacseq-marmoset-subcortex

![ATACseq Marmoset Subcortex screenshot](/assets/papers/HMBAViewers/ATAC_Viewer.webm)

**Project:** [HMBA:Basal Ganglia Preprint/ATAC-seq + Spatial Tx Marmoset Subcortex](https://allen-hmba-releases.s3.amazonaws.com/cytosplore/Release/BasalGanglia_Preprint/Preprint_2025_ATACseq_Marmoset_Subcortex.mv) in the [Cytosplore Viewer](/#get)

**Contact:** Chang
**Email:** c.li1@lumc.nl

**Description:** This viewer enables interactive exploration of ATAC-seq and RNA-seq data, linking peaks to genes via matching annotations. TAC-seq peaks and RNA-seq genes are projected onto the spatial domain based on matching annotations. RNA-seq genes with spatial patterns similar to a selected ATAC-seq peak (and vice versa) can be identified. The top cell types associated with these features can be inspected and displayed on the spatial map and the RNA-seq UMAP.
