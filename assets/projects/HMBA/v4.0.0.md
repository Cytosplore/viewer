**Email:** j.g.l.thijssen@lumc.nl

**Description:** This project provides interactive, linked visualizations of Macaque Basal Ganglia Patch-seq data gathered by the HMBA project. It integrates three single-cell modalities—transcriptomics, electrophysiology, and morphology—enabling users to explore how these features relate to one another and uncover what makes different cell types distinct.
