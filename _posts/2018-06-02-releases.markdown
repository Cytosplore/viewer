Here you can find the releases and change notes of Cytosplore MTG Viewer. If you need an older build for reproducing previous results please download them below.

**Current Version: 1.0**  
[<i class="fab fa-windows"></i> Windows 7 and higher (64bits)](https://link/install_cytosplore.exe)  

During the first run Cytosplore MTG Viewer will automatically check for the latest datafile and download it. If you want to install Cytosplore MTG viewer on multiple computers or a computer without internet access you can download the datafile separately and unpack the data.hdf file in the same directory as the Cytosplore MTG.exe.  
[Datafile](https://link/data.zip)
